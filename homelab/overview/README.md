# Homelab Overview

My homelab is a personal environment used to simulate enterprise IT and security infrastructure for hands-on learning in Cyber Operations and Systems Administration.

## Purpose
- Practice system administration
- Simulate enterprise identity and access management
- Experiment with security controls
- Learn network and system behavior in a controlled environment

## Core Components (Example Setup)

- Windows Server (Active Directory domain services)
- Windows client machines
- Linux server (Ubuntu)
- Networking equipment (router/firewall simulation)
- Virtualization platform (VirtualBox / VMware)

## Learning Focus Areas
- Identity and Access Management (Active Directory concepts)
- Network segmentation and traffic flow
- Logging and monitoring concepts
- System hardening practices
- Basic attack and defense simulation (controlled lab use)

## Why It Matters
This environment allows me to connect theoretical cybersecurity concepts to real operational systems, reinforcing both technical and governance-level understanding.
